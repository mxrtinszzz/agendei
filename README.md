# Sistema de Agendamento de Serviços (React Native)

## Descrição do Projeto
Este aplicativo foi desenvolvido utilizando React Native com o objetivo de simular um sistema de agendamento de serviços, como consultas médicas, serviços de estética e atendimentos diversos.

A aplicação permite que o usuário navegue entre categorias, explore estabelecimentos, visualize serviços disponíveis e realize agendamentos de forma simples e intuitiva.

## Funcionalidades

O aplicativo conta com as seguintes telas implementadas:

- Splash (Tela Inicial)  
  Tela de carregamento exibida na inicialização do aplicativo.

- Login  
  Tela de autenticação com validação simulada.

- Home  
  Exibição de categorias de serviços disponíveis.

- Explorar  
  Lista de estabelecimentos e profissionais com dados mockados.

- Reservas  
  Tela onde o usuário pode visualizar seus agendamentos.

- Perfil  
  Exibição de informações do usuário.

- Estabelecimento  
  Detalhes do estabelecimento, incluindo serviços oferecidos.

- Agendamento  
  Seleção de data e horário para realização de reservas.

## Tecnologias Utilizadas

- React Native
- React Navigation
- JavaScript

## Implementação Técnica

- Navegação entre telas implementada com React Navigation
- Utilização de dados mockados para simular respostas de API
- Estrutura organizada por pastas para facilitar manutenção
- Layouts desenvolvidos com foco em consistência visual
- Interface responsiva para diferentes tamanhos de tela

## Interface e Design

A interface foi construída seguindo uma identidade visual consistente, utilizando a seguinte paleta de cores:

- Verde: #5cc6ba
- Cinza escuro: #717f7f
- Branco: #ffffff
- Tons neutros para apoio

Foram utilizados ícones e elementos visuais para melhorar a experiência do usuário, mantendo clareza e simplicidade em todas as telas.
